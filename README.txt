Программа была создана Yeskozha Syrym from Astana IT University
Были использованы:
Node.js, EJS, Document Object Model, HTML, JavaScript, CSS, Bootstrap

Чтобы запустить программу:
1) Скачать ZIP файл и расархивировать
2) Открыть командную строку(Win+R и ввести cmd)
3) Указать путь к файлу(На Windows это: cd "C:\Users\User\Downloads\endterm_project-main\endterm_project-main")
4) Установить библиотеки Node.js(ввести в командную строку: npm install)
5) Запускаете локальный сервер(node server.js)
6) Переходите на локальный сервер в любом браузере(localhost:5000)
7) Готово
